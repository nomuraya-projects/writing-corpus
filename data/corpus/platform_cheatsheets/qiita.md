## 見出し

```markdown
# 見出し1
## 見出し2
### 見出し3
```

---

## 強調・装飾

| 記法 | 結果 |
|------|------|
| `*イタリック*` | *イタリック* |
| `**太字**` | **太字** |
| `~~打ち消し~~` | ~~打ち消し~~ |
| `` `code` `` | `code` |

---

## 注釈ボックス（独自記法）

```markdown
:::note info
インフォメーション
:::

:::note warn
警告
:::

:::note alert
アラート
:::
```

## コードブロック

````markdown
```ruby:ファイル名.rb
puts "ファイル名表示"
```

```diff
- 削除行
+ 追加行
```
````

シンタックスハイライト対応言語多数
